# Projeto-Front-End-ADS-FIAP
Atividade Fase 04 - Curso de Análise e Desenvolvimento de Sistemas FIAP 

Este trabalho apresenta o desenvolvimento da tela de cadastro do projeto FIAP Bank, criada inicialmente na atividade da Fase 02 utilizando o Figma. A tela foi recriada utilizando HTML, CSS e Tailwind CSS, seguindo a identidade visual definida no protótipo e aplicando conceitos de design responsivo para diferentes tamanhos de tela.

Entre as quatro telas elaboradas no protótipo, foi selecionada a tela de cadastro por representar o primeiro contato do usuário com a plataforma, permitindo demonstrar a estrutura visual, a organização dos formulários e a experiência de navegação proposta para o sistema Fintech.

A interface foi desenvolvida com foco em usabilidade, acessibilidade e consistência visual, mantendo as cores, tipografia e elementos gráficos definidos no projeto original.
